# Summary

OKF is an open, human- and agent-friendly format for representing knowledge:
a directory of markdown files with YAML frontmatter. Only `type` is
required in frontmatter; `title`, `description`, `resource`, `tags`, and
`timestamp` are recommended. Reserved filenames `index.md` (directory
listing) and `log.md` (update history) have defined structure.
